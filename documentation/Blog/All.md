# All blogs

Used to get all blogs

**URL** : `/api/blog/`

**Method** : `GET`

**Auth required** : NO

## Success Response

**Code** : `200 OK`

**Content example**

```json
[
  {
    "title": "string",
    "body": "string",
    "user_id": 0,
    "created_at": "2022-10-26T19:16:38.743Z",
    "updated_at": "2022-10-26T19:16:38.743Z",
    "comments": []
  }
]
```
